## Image

### Using Markdown

The <a href="https://www.markdownguide.org/basic-syntax/#images" target="_blank" rel="noopener noreferrer">markdown syntax</a> lets you add images using the following code

```md
![title](/path/image.jpg)
```

Note that the image file size must be less than 5MB. Otherwise, we recommend hosting on a service like <a href="https://cloudinary.com/" target="_blank" rel="noopener noreferrer">Cloudinary</a> or <a href="https://aws.amazon.com/s3/" target="_blank" rel="noopener noreferrer">S3</a>. You can then use that URL and embed.

### Using embeds

To get more customizability with images, you can also use [embeds](/writing-content/embed) to add images

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Loads another HTML page within the document. Most commonly used for embedding videos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
