## Welcome to RevenueBase

RevenueBase helps you validate email addresses and work with company data via API and data feeds. Use this doc set to get up and running quickly and to explore the full API and data offerings.

## What you can do

- **API** — Real-time email verification, batch processing, and company resolution. See the [API Reference](/api-reference/authentication) for authentication and all endpoints.
- **Data Feeds** — Bulk and streaming data with documented schemas. Start with the [Data Feeds overview](/data-feeds/overview) and [Tables & Schema](/data-feeds/tables-schema).
