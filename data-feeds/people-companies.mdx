The `people_companies` view combines every contact record with its associated company firmographic data in a single flat table. No joins required — just query directly.

## What's included

This pre-joined table combines person and company data. No joins needed — query directly from the table.

Every person field plus every company field, already joined for you.

| Metric | Value |
|--------|-------|
| **Total records** | One row per contact (see [Contacts](/data-feeds/contacts) for counts) |
| **Primary key** | `RBID_PAO` |
| **Source tables** | `PER_LATEST` LEFT JOIN `ORG_LATEST` on `RBID_ORG` |

## Data dictionary

Field names, descriptions, and examples are sourced from the [Data dictionary](/data-feeds/data-dictionary). This pre-joined table includes all person fields (suffix `_PER`) and all company fields (suffix `_ORG`).

### Identifiers

| Field | Description | Example |
| --- | --- | --- |
| `RBID_PER` | RevenueBase's unique identification for the person. The RBID stays the same if someone changes their job. | rb-pabq6o2xf |
| `RBID_ORG` | RevenueBase's unique identifier for a company | rb-oab75n3pe |
| `RBID_PAO` | RevenueBase's unique identifier for the person at a specific company. | Revenue Base Internal field |

## Example queries

### Prospect list — no joins needed

```sql
SELECT
    FIRST_NAME,
    LAST_NAME,
    EMAIL_ADDRESS,
    JOB_TITLE,
    COMPANY_NAME_ORG,
    LINKEDIN_INDUSTRY_ORG,
    EMPLOYEE_COUNT_MAX_ORG,
    HEADQUARTERS_STATE_NAME_ORG
FROM RELEASES.RELEASE.VELOCITY_BASE_UNLIMITED_LATEST
WHERE JOB_LEVEL IN ('VP', 'Director', 'C-Suite')
  AND LINKEDIN_INDUSTRY_ORG LIKE '%Software%'
  AND EMPLOYEE_COUNT_MAX_ORG BETWEEN 100 AND 1000
  AND HEADQUARTERS_COUNTRY_NAME_ORG = 'United States'
ORDER BY EMAIL_LAST_VERIFIED_AT DESC
LIMIT 500;
```

### Export for a campaign

```sql
SELECT
    FIRST_NAME,
    LAST_NAME,
    EMAIL_ADDRESS,
    JOB_TITLE,
    COMPANY_NAME_ORG,
    EMPLOYEE_COUNT_RANGE_ORG,
    LINKEDIN_INDUSTRY_ORG,
    HEADQUARTERS_CITY_ORG,
    HEADQUARTERS_STATE_NAME_ORG
FROM RELEASES.RELEASE.VELOCITY_BASE_UNLIMITED_LATEST
WHERE JOB_FUNCTION = 'Marketing'
  AND EMPLOYEE_COUNT_MAX_ORG >= 200
  AND EMAIL_LAST_VERIFIED_AT >= DATEADD(day, -30, CURRENT_DATE())
ORDER BY COMPANY_NAME_ORG, LAST_NAME;
```
